How to Compile: run:
>> make all

How to run: enter
>> java Ethernet seed N #frames arrival_rate

where seed is a seed to use for random numbers (0 for random sed)
N is the number of devices to simulate
#frames is the number of frames to simulate
arrival_rate is the mean arrival rate of frames per 100 slot times
